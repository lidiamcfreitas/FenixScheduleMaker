                                                                              It's a mistake in the creation of the stored proc. It only accepts 20 characters in each parameter:
                                                                              Change the parameters to accept 255 characters each:

                                                                              CREATE DEFINER=`root`@`localhost` PROCEDURE `sp_createUser`(

                                                                              IN p_name VARCHAR(255),

                                                                              IN p_username VARCHAR(255),

                                                                              IN p_password VARCHAR(255)

                                                                              )
